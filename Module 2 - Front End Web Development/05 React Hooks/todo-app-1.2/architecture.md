Libraries :

1. React
2. Vite
3. DnD
4. Tailwind

User Stories :

1. User can change the theme from the dark mode to light mode and vice versa.
2. User can add new to do list.
3. User can search the to do list
4. User can completed or uncompleted the to do list by checking the to do list.
5. User can deleted the to do list.
6. User can drag and drop the to do list.
7. User can filter the status of to do list.
8. User can clear the completed to do list.
9. User can view the number of the active items.
10. User can sort the to do list.

Data Structure :

1. Theme : boolean
2. New text input: String
3. Search input : String
4. Todo : array (todoitem string)
5. todo status: boolean
6. Filter status : union types
7. Sort : union types
